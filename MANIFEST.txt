本地验证步骤(与容器内一致)
================================================================
后端(Go 1.25.1，go.mod 要求)：
  export CGO_ENABLED=0
  # 无嵌入前端也可编译校验：
  go build -tags noembed ./...
  go test  -tags noembed ./...
  # 正式(内嵌前端)构建需先构建前端产物到 web/classic/dist：
  go build ./...
  go build -ldflags="-s -w" -trimpath -o new-api-lite .

前端(项目优先 Bun；无 Bun 时用 npm)：
  cd web/classic
  bun install && bun run build
  # 或(无 bun)：
  npm install --legacy-peer-deps          # 依赖存在 peer 冲突，需 --legacy-peer-deps
  NODE_OPTIONS=--max-old-space-size=3584 npx vite build   # 构建内存需 >2GB，故调大堆
